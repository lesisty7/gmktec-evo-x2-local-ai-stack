# Security Policy

This repository contains public documentation, example configurations, and helper scripts for a local AI stack.

It is not a hosted service and it does not provide security guarantees for your own deployment.

## Scope

The examples in this repository are designed to be:

- sanitized
- documentation-first
- adaptable to other environments

They are not intended to be copied into production without review.

## If You Find a Security Problem

If you discover a security issue in one of the published examples or scripts:

- avoid posting real secrets, tokens, cookies, or private infrastructure details
- describe the issue with sanitized reproduction details
- report it through GitHub in the least sensitive way available to you

If the issue would expose private or dangerous information when disclosed publicly, prefer a private reporting path if one is available.

## Safe Usage Expectations

Before using any example from this repository:

- replace all placeholder credentials
- review all network addresses, hostnames, and ports
- review all mounted paths and storage assumptions
- validate the security model for your own LAN or WAN exposure

## No Warranty

This repository is provided as-is.

Examples may be incomplete, conservative, outdated, or unsuitable for your environment without modification.
