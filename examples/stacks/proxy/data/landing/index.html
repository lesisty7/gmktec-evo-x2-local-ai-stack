Local AI Gateway

This is a neutral landing page for the example reverse-proxy stack in this repository.

The proxy example is intentionally small. It demonstrates one reasonable pattern for exposing local AI services through a single gateway without publishing a private routing map.

  • llama.ai.local for the local LLM backend
  • openwebui.ai.local for the main web UI
  • searxng.ai.local for search
  • comfyui.ai.local for image generation
